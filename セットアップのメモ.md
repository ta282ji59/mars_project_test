## スーパーユーザー作成
```bash
$ docker-compose exec django 
$ python manage.py createsuperuser
```
（出力）

メールアドレス: ta282ji@icloud.com<br>
Password: <br>

## データベース
- コンテナを起動
```bash
docker exec -it redace_db bash
```
- データベースに接続
```bash
psql -U ta282ji -d mars
```
- テーブルにアクセス
```bash
select * from CRISM
```
